# B26 Ex01 Collaboration Workspace

This repository is the working area for the pair assignment before the final course submission package is prepared.

## Current Structure

- `Ex01_01` is the current C# project skeleton.
- `Professor_Rulebook` contains the translated lecturer rules and workflow notes.
- `Professor_Exe_Example` contains the lecturer example executable and access details.
- `RulePages` contains exported image pages from the lecturer materials.

## Current Technical State

- The current project targets `.NET Framework 4.7.2`.
- The current `Program.cs` is still only a minimal entry point.
- The current solution file is a blank `.slnx`, so the solution setup should be verified in Visual Studio before real development continues.

## Recommended Collaboration Flow

1. Keep `main` stable and always buildable.
2. Open a short branch for each task, such as `feature/login`, `feature/binary-series`, or `feature/sand-clock`.
3. Merge small changes frequently instead of keeping long-running branches.
4. Pull before starting work and before pushing work.
5. Use commit messages that describe one clear change each.

## Important Course Reminder

GitHub is for development only. Before the final course submission, remove generated folders and the `.git` folder from the packaged hand-in, according to the lecturer submission rules.
